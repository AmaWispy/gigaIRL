# Система данжей gigaIRL

Документ описывает вход в данж, прохождение по этажам, лут, награды и каталог данжей по тирам. Экипировка и сеты — в [equipment-system.md](equipment-system.md).

---

## 1. Суть данжа

Данж — **отдельный инстанс из 10 этажей**. Игрок тратит энергию на вход и перемещение, сражается с мобами (без победы дальше не пройти), собирает ресурсы и сокровища, на 5-м этаже всегда встречает **редкого моба**, на 10-м — **босса**. За полное прохождение выдаётся **Печать исследователя** (пока без применения — задел на будущий контент).

| Параметр | Значение |
|----------|----------|
| Этажей | **10** |
| Вход | **10 ⚡** |
| Переход на следующий этаж | **10 ⚡** |
| Полное прохождение (вход + 9 переходов) | **100 ⚡** минимум |
| Бой с мобом | **1 ⚡** (как в открытом мире) |

HP в данже **не восстанавливается между этажами** (кроме зелий и дропа). Поражение в бою — **вылет из данжа** с сохранением уже полученного лута.

---

## 2. Два способа попасть в данж

### 2.1. Пропуск в данж (500 💰)

- Покупается **один раз за тир** у торговца в городе тира (или в гостинице/оружейнике).
- После покупки данж **всегда доступен** из меню «Данжи» / точки входа в городе — не нужно искать в поле.
- Цена: **500 золота** за пропуск тира 1; далее по тирам (см. таблицу в §8).

Предмет: `dungeon_pass_t{N}` (не расходуется, флаг в инвентаре или `character_dungeon_unlocks`).

### 2.2. Бесплатный вход при исследовании

- На **каждом тире** в открытых зонах при «Осмотреться» может выпасть событие **«Вход в данж»** (редкое действие, ⚡ 2).
- Подбирает **данж текущего тира** (игрок в Тёмном Лесу тира 1 → Заброшенная Шахта).
- Вход в инстанс **бесплатный по золоту**, но **10 ⚡** на пороге данжа всё равно тратится при принятии.
- Если пропуск уже куплен — событие всё равно может выпасть как «бесплатный телепорт к входу» (без доп. золота).

| Способ | Золото | Энергия на порог | Повторяемость |
|--------|--------|------------------|---------------|
| Пропуск | 500 💰 один раз | 10 ⚡ за забег | Вход всегда открыт |
| Исследование | 0 | 10 ⚡ за забег | Случайно при осмотре |

## 3. Структура этажа

При генерации забега для каждого этажа `1…10` создаётся **обязательная** угроза и **опциональные** бонусы.

### 3.1. Фиксированные этажи

| Этаж | Обязательно | Примечание |
|------|-------------|------------|
| **1–4** | Обычный моб данжа | Без победы переход заблокирован |
| **5** | **Редкий моб** | Всегда; усиленный лут и XP |
| **6–9** | Обычный моб данжа | Как 1–4 |
| **10** | **Босс данжа** | Финал; лучший лут, печать за весь данж |

### 3.2. Случайные события на этаже

Помимо обязательного моба (кроме этажей, где моб = редкий/босс), на этаже **независимо** могут появиться:

| Событие | Шанс (за этаж) | Действие |
|---------|----------------|----------|
| **Скопление ресурсов** | 35% | Сбор без боя, ⚡ 1 |
| **Сокровище** | 20% | Открыть сундук, ⚡ 1 |

- На одном этаже может быть **и моб, и ресурс, и сокровище**.
- Сначала игрок выбирает порядок; **к мобу обязателен** перед уходом с этажа.
- Ресурсы и сокровища **необязательны**.

### 3.3. Блокировка перехода

```
Можно уйти на этаж N+1  ⇔  обязательный моб этажа N побеждён
```

Кнопка «Следующий этаж (10 ⚡)» неактивна, пока флаг `floor_cleared = false`.

## 4. Лут

### 4.1. Ресурсы

- Количество **выше**, чем при обычном сборе в поле (×2–×3 к базе).
- Плюс шанс **редких данжевых ресурсов** — **пока не используются в крафте** (инвентарь + продажа торговцу).

| Тир | Обычный дроп (примеры) | Редкий данжевый (не для крафта) |
|-----|-------------------------|----------------------------------|
| 1 | 🟢 руда, кожа, древесина ×2–4 | Обсидиановый осколок, Пыль рун |
| 2 | 🔵 шкура, кристалл ×2–3 | Ядовитый секрет, Хитиновая пластина |
| 3 | 🔵+🟣 ресурсы | Сердцевина циклопа, Осколок зеркала |
| 4 | 🟣+🔴 ресурсы | Чешуя дракона, Слеза вулкана |

Шанс редкого данжевого ресурса: **15%** с кучи ресурсов, **8%** с обычного моба.

### 4.2. Сокровища

- Деньги: **×1.5** к обычному сокровищу в поле (`random 10–50` → в данже `15–75` на тир 1, масштаб по тиру).
- Дополнительно те же таблицы, что у мобов (печати, сферы — см. equipment-system), с пониженным шансом.

### 4.3. Зелья лечения

| Источник | Шанс |
|----------|------|
| Обычный моб данжа | 5% |
| Редкий моб (этаж 5) | 8% |
| Босс (этаж 10) | 10% |
| Сокровище | 7% |

Восстановление: как у обычного зелья (+30 HP или по `config`).

### 4.4. Экипировка (данжевые сеты)

Качество при дропе: ⚪ 40% / 🟢 35% / 🔵 18% / 🟣 5% / 🔴 2% (как в equipment-system).

| Источник | Шанс предмета сета | Примечание |
|----------|-------------------|------------|
| Обычный моб | 3% | Случайный предмет сета; 70% шанс — не оружие (одежда) |
| Редкий моб (5) | 12% | |
| Босс (10) | **35%** гарантированно **1** предмет + 25% на второй | |
| Сокровище | 5% | |

Подробности сетов — [equipment-system.md §6](equipment-system.md).

### 4.5. Опыт

Опыт за бой в данже **значительно выше** открытого мира (ориентир ×2.5 к мобу той же зоны):

| Тип | Множитель к XP открытого мира |
|-----|-------------------------------|
| Моб данжа (этажи 1–4, 6–9) | ×2.0 |
| Редкий (этаж 5) | ×3.5 |
| Босс (этаж 10) | ×6.0 |

Деньги с мобов: ×1.5 к полевым аналогам.

### 4.6. Награда за полное прохождение

При победе над боссом на 10-м этаже:

| Награда | Количество |
|---------|------------|
| **Печать исследователя** | 1 (пока не тратится; задел на косметику/доступ/крафт) |
| Бонус XP | +50% от суммы XP за забег |
| Бонус 💰 | +100 золота (масштаб по тиру) |

Предмет: `explorer_seal` — `type: dungeon_token`.

## 5. Поток игрока (UI)

```
[Город / Исследование]
        │
        ├─ Купить пропуск (500💰) ──► Данж всегда в меню
        │
        └─ Событие «Вход в данж» ──► Подтверждение (10⚡)
                │
                ▼
        [Экран данжа: этаж 1/10, HP, энергия]
                │
                ├─ События этажа (моб / ресурсы / сокровище)
                ├─ Бой (с пропуском анимации)
                │
                └─ «Следующий этаж» (10⚡) ──► … ──► Этаж 10, босс
                                        │
                                        ▼
                              [Победа: печать + итог лута]
                              [Поражение: вылет, лут сохранён]
```

- Выйти добровольно можно с любого этажа **после победы над мобом** (кнопка «Покинуть данж»).
- При выходе до босса печать исследователя **не выдаётся**.

## 6. Связь с тирами мира

| Тир | Уровень игрока | Открытые зоны (пример) | Данж | Сет |
|-----|----------------|------------------------|------|-----|
| **1** | 1–7 | Золотое Поле, Тёмный Лес | Заброшенная Шахта | Шахтёрский гнев (ур. 7) |

## 9. Шансы при исследовании (бесплатный вход)

Событие добавляется в пул редких действий при осмотре в зоне **своего тира**:

| Тир | Шанс «Вход в данж» среди 2–4 действий |
|-----|---------------------------------------|
| 1 | ~8% на каждый слот действия |
| 2 | ~6% |
| 3 | ~5% |
| 4 | ~4% |

Не чаще **1 раза в день** на персонажа (опционально, анти-абуз).

## 10. Техническая реализация (план)

### 10.1. Модели

| Таблица | Назначение |
|---------|------------|
| `dungeons` | Шаблон: имя, тир, `set_key`, `item_level`, `min_power` |
| `dungeon_monsters` | `dungeon_id`, `role` (normal/rare/boss), ссылка на `monsters` или inline-статы |
| `dungeon_runs` | Активный забег: `character_id`, `dungeon_id`, `current_floor`, `hp_snapshot`, `is_active` |
| `dungeon_floor_states` | `run_id`, `floor`, `mob_defeated`, `resources_claimed`, `treasure_claimed` |
| `character_dungeon_unlocks` | `character_id`, `dungeon_id` (купленный пропуск) |

### 10.2. Сервисы

- `DungeonService::startRun(character, dungeon)` — 10 ⚡, создание run.
- `DungeonService::advanceFloor(run)` — 10 ⚡, генерация событий этажа.
- `DungeonService::resolveMob(run)` — бой через `CombatService`, без траты энергии на бой в данже (или 1 ⚡ — на выбор; в GDD выше: 1 ⚡).
- `DungeonService::claimResource / claimTreasure`.
- `DungeonService::completeRun(run)` — печать исследователя, закрытие run.
- `DungeonService::abandonOnDefeat(run)` — сохранение лута, сброс run.

### 10.3. Предметы (сидер)

```text
dungeon_pass_t1 … t4
explorer_seal
obsidian_shard, rune_dust, … (редкие ресурсы по тирам)
```

### 10.4. UI

- `Game/Dungeon/Index` — список данжей (открыты пропуском).
- `Game/Dungeon/Run` — этаж, карточки событий, HP, кнопка «Следующий этаж».
- Торговец / гостиница: покупка пропуска.
- Исследование: новый `ExplorationActionType::DungeonEntrance`.

### 10.5. Фазы разработки

| Фаза | Содержание |
|------|------------|
| **F1** | Документ, `config/dungeon.php`, модели, Шахта тир 1, пропуск 500💰 |
| **F2** | Забег 10 этажей, блокировка без победы, лут ресурсов/сокровищ |
| **F3** | Дроп сета Шахты, зелья, XP, печать исследователя |
| **F4** | Событие входа при исследовании |
| **F5** | Данжи тиров 2–4, редкие ресурсы, баланс |

### 10.6. Config (черновик)

```php
// config/dungeon.php
return [
    'entry_energy' => 10,
    'floor_energy' => 10,
    'floors_total' => 10,
    'rare_floor' => 5,
    'boss_floor' => 10,
    'pass_prices' => [1 => 500, 2 => 800, 3 => 1200, 4 => 2000],
    'resource_pile_chance' => 35,
    'treasure_chance' => 20,
    'potion_drop' => ['normal' => 5, 'rare' => 8, 'boss' => 10, 'treasure' => 7],
    'equipment_drop' => ['normal' => 3, 'rare' => 12, 'boss' => 35, 'treasure' => 5],
    'xp_multiplier' => ['normal' => 2.0, 'rare' => 3.5, 'boss' => 6.0],
    'explorer_seal_on_clear' => 1,

```

## 12. Сводка для игрока (кратко)

- Купи **пропуск за 500💰** — данж всегда открыт, или найди **вход при осмотре** бесплатно.
- **10 этажей**, вход и каждый переход — **10 ⚡**.
- На каждом этаже **нужно победить моба**; на 5-м — редкий, на 10-м — босс.
- По пути — **ресурсы** (много + редкие), **сокровища**, **зелья**, **данжевый шмот**.
- За весь данж — **Печать исследователя** и много **опыта**.
